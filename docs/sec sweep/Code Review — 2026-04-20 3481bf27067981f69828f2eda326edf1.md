## Issues Found

### BUG

**`src/components/GameCard.tsx` — Bundled-online banner bypasses i18n**

The warning banner added in `348edd2` uses two hardcoded English strings:

```
"Bundled online mode — additional risk"
"The LumiScore above reflects the single-player campaign only."
```

Every other string in this component goes through `t()` from `next-intl`. These will show English on all locales. Fix: add `bundledOnlineTitle` and `bundledOnlineCopy` keys to the `gameCard` messages namespace and replace the hardcoded strings with `t('bundledOnlineTitle')` and `t('bundledOnlineCopy')`.

### IMPROVEMENT

**`src/app/[locale]/page.tsx` — Roblox carousel missing `thumbnailUrl` filter**

`238bca4` correctly pushed `isNotNull(platformExperiences.thumbnailUrl)` into SQL for the Fortnite query (was silently returning 1 result instead of 8). The Roblox query fetches without the same guard. Confirm `RobloxCarouselRow` handles null thumbnails gracefully, or apply the same `isNotNull` filter.

---

**`src/app/api/cron/debate-games/route.ts` — Gemini response unvalidated (`as any`)**

```tsx
const a = part.functionCall.args as any
```

The returned scores go straight into `db.update()` without validation. If Gemini returns a malformed response, corrupt scores land in the database silently. Fix: parse with Zod before trusting the payload — a `z.record(z.string(), z.number())` per score group catches the most common failure modes.

---

**`fortnite-discovery-raw.json` committed to repo (`fae16fc`)**

A raw API response file was committed alongside the import script. Such files are ephemeral, grow over time, and accumulate in git history. Add `fortnite-discovery-raw.json` (or `*-raw.json`) to `.gitignore`.

## Patterns Worth Addressing

### Push-fix regression cycle

Three commits were needed to ship the Plus Jakarta Sans logo without breaking dark mode: add font → dark mode breaks → revert body class → still broken → remove `next/font` entirely. The `next-themes` + `next/font` hydration conflict is a known footgun. A quick pre-push check (toggle dark mode in dev before pushing any layout/font change) would catch this class of regression before it hits `main`.

Similarly, `bundledOnlineNote` took three commits to propagate (`348edd2` → `b3d2d25` → `b433eb8`) — the DB column and component landed together but the review-page and API-route serializations were missed. A grep for the field name across `types/`, all `route.ts` files, and `page.tsx` before committing closes the loop.
